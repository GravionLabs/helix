# DynamicDialog

> Service-based dialogs: open any component in a dialog at runtime via `DialogService`.

## Import

```ts
import { DynamicDialog, DynamicDialogContent } from '@gravionlabs/helix/dynamicdialog';
```

## Components

### DynamicDialog

Selector: `h-dynamicDialog, h-dynamicdialog, h-dynamic-dialog`

### DynamicDialogContent

Selector: `[hDynamicDialogContent]`

## Services

### DialogService

Dynamic Dialog component methods.

## Interfaces & Types

- `DynamicDialogTemplates` — Defines valid templates in Dynamic Dialog.
